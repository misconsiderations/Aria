Privacy Policy

Last updated April 2026 · How we handle your data.

Aria stores account and dashboard operational data required for hosting and moderation. This policy explains what is collected and how it is used.

Data Collected

Login account metadata (user ID, username, hashed password), hosted-user records (Discord token references, per-user settings), instance activity logs, and dashboard configuration.

Data Access

Owner/admin accounts can view operational logs and hosted account summaries for management and abuse prevention. Individual user data is not shared with other users.

Data Retention

Logs and account records may be retained for stability and security purposes. Removal is at owner/admin discretion or upon explicit user request.

Security

Passwords are stored as PBKDF2-SHA256 hashes with per-user salts. Bot tokens are stored in encrypted user records and are never exposed in dashboard responses.
