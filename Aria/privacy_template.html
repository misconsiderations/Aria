Privacy

How Aria handles operator and runtime data.

The panel keeps only the operational data needed to authenticate users, link instances, display history, and enforce access. This page exists to make that boundary explicit.

Policy route operational storage only
Scope Panel
Applies to dashboard accounts, access requests, hosted links, runtime history, and system metrics.

Data Types

Aria stores dashboard account metadata, hosted configuration, request records, activity timeline entries, and runtime analytics needed for service operation.

Access Controls

Role-based restrictions are enforced across dashboard routes and APIs so users only access allowed resources.

Credential Security

Dashboard passwords are stored as one-way hashes and sensitive values are not exposed in UI responses.

Retention and Auditing

Operational logs and selected records may be retained for abuse prevention, reliability, and troubleshooting.
